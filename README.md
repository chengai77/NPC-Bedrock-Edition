## 功能特性

### 核心功能
- **NPC生成蛋**: 在创造模式物品栏中可找到"NPC生成蛋"，生成原版NPC
- **编辑功能**: 创造模式玩家右键NPC可以编辑所有属性
- **对话系统**: 生存/冒险模式玩家右键NPC可以对话
- **指令执行**: NPC可以执行预设的指令
- **自定义皮肤**: 支持100种预设皮肤，可自行替换
- **AI寻路功能**: NPC可以自主行走或站立不动

### 皮肤预设系统
- **npc_1**: 作者皮肤 - 名称自动设置为「作者」，**名称锁定不可更改**
- **npc_2**: 星野皮肤 - 名称自动设置为「星野」
- 其他皮肤: 可自定义名称

### 编辑功能（创造模式右键NPC）
1. **编辑名称**: 修改NPC的显示名称（部分皮肤名称锁定）
2. **编辑对话**: 添加、编辑、删除对话内容
3. **编辑指令**: 添加、编辑、删除可执行的指令
4. **选择皮肤**: 从100种预设皮肤中选择，自动更新名称
5. **AI寻路**: 开启/关闭NPC自主行走功能
6. **测试功能**: 测试对话和指令效果
7. **删除NPC**: 删除不需要的NPC

### 对话系统（生存/冒险模式右键NPC）
- 与NPC对话时会随机显示对话内容
- 支持多个回复选项
- 对话时可触发关联的指令

### AI寻路功能
- **开启AI**: NPC会在附近自主行走，模拟真实行为
- **关闭AI**: NPC站立不动，保持原位

## 使用方法

### 生成NPC
1. **生成蛋**: 在创造模式物品栏中找到"NPC生成蛋"
   - 右键放置生成原版NPC (minecraft:npc)
2. **命令**: 使用命令 `!npc spawn`

### 编辑NPC（创造模式）
1. 右键点击NPC打开编辑菜单
2. 选择要编辑的内容
3. 按照提示输入信息
4. 保存更改

### 与NPC对话（生存/冒险模式）
1. 右键点击NPC
2. 选择回复选项
3. NPC会执行关联的指令

### 聊天命令
- `!npc spawn` - 生成一个NPC
- `!npc list` - 列出所有NPC
- `!npc help` - 显示帮助信息

## 皮肤系统

### 预设皮肤
- 共100种预设皮肤
- 文件名格式: `npc_1.png`, `npc_2.png`, ..., `npc_100.png`
- 在编辑NPC时可预览和选择

### 已配置皮肤
| 皮肤ID | 文件名 | 名称 | 锁定状态 |
|--------|--------|------|----------|
| 1 | npc_1.png | 作者 | ✅ 锁定 |
| 2 | npc_2.png | 星野 | ❌ 可更改 |

### 自定义皮肤
1. 解压资源包
2. 找到 `textures/entity/npc_skins/` 目录
3. 替换对应的皮肤文件
4. 重新打包资源包

## AI寻路配置

在 `BP/scripts/main.js` 中可以调整AI行为参数：

```javascript
const AI_CONFIG = {
    walkSpeed: 0.1,           // 行走速度 (0.05-0.2)
    wanderRadius: 10,         // 游荡半径 (5-20格)
    wanderInterval: 100,      // 游荡间隔 (50-200 ticks)
    pauseChance: 0.3,         // 暂停概率 (0.0-1.0)
    pauseDuration: [50, 150]  // 暂停时长范围 (ticks)
};
```

## 技术规格

- **版本**: 1.26.10 (Tiny Takeover)
- **Script API**: @minecraft/server v2.6.0 (稳定版)
- **NPC类型**: 原版NPC (minecraft:npc)
- **交互方式**: 右键点击 (playerInteractWithEntity)
- **作者**: 承挨

## 注意事项

1. **交互方式**: 创造模式右键编辑，生存/冒险模式右键对话
2. **皮肤文件**: 必须是64x64像素的PNG格式
3. **指令权限**: NPC执行的指令需要游戏权限支持
4. **性能考虑**: 避免创建过多NPC影响游戏性能
5. **AI寻路**: 开启AI后NPC会自主行走，注意不要设置过大的游荡半径
